# Personal Portfolio

A high-performance personal portfolio built with Next.js, TypeScript, and Tailwind CSS to showcase software engineering projects, technical skills, and competitive programming profiles.

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Framer Motion
* React Icons
* Vercel

## Architecture & Features

* Minimalist, responsive interface optimized for technical recruiters.
* Centralized data management under `src/data/` for projects, skills, education, competitive programming profiles, and social links.
* Easily maintainable resume integration through a persistent external link.
* Component-driven architecture for modularity and scalability.
* Smooth, lightweight animations with minimal runtime overhead.
* Responsive layout across desktop, tablet, and mobile devices.
* Optimized for performance, accessibility, and maintainability.
* Automated deployment through Vercel with continuous integration on every push.

## Local Development

```bash
npm install
npm run dev
```

## Deployment

The application is deployed on Vercel using an automated CI/CD pipeline. Every commit pushed to the configured branch automatically triggers a production build and deployment.
